# Learn Python for Data Analysis

This repository contains structured learning content to understand **Python for Data Analysis** from basics to real-world projects.

---

## 📘 Course Structure

### Phase 1: Python Basics

* Variables & Data Types
* Data Structures (List, Tuple, Set, Dictionary)
* Control Flow (if-else, loops)
* Functions
* File Handling

### Phase 2: NumPy

* Arrays (1D, 2D)
* Indexing & Slicing
* Mathematical Operations
* Aggregations (mean, sum, etc.)
* Reshaping & Random Data

### Phase 3: Pandas

* Series & DataFrame
* Data Cleaning
* Filtering & Selection
* GroupBy & Aggregation
* CSV Read/Write

### Phase 4: Data Visualization

* Line, Bar, Histogram, Scatter Plots
* Seaborn Visualizations
* Heatmaps & Pairplots

### Final Project

* Airbnb Dataset Analysis
* Data Cleaning, Analysis, Visualization
* Business Insights

---

## 📂 Repository Contents

* Python notebooks/scripts for each phase
* Practice tasks and examples
* Real-world project (Airbnb analysis)
* Interview questions for practice
* Index document (Word file) for quick reference

---

## 🚀 How to Use

1. Clone the repository
2. Open files using Jupyter Notebook
3. Follow phase-wise learning
4. Practice tasks in each section
5. Complete the final project

---

## 🎯 Objective

* Build strong Python fundamentals
* Learn data analysis using NumPy and Pandas
* Create visual insights using Matplotlib and Seaborn
* Solve real-world data problems
